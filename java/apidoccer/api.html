Properties

This is a list of slaves. Access the elements using the key
/status/slaves
The host name of the slave
/status/slaves/${key}/host
The port on which the agent is listening
/status/slaves/${key}/agentPort
The slave system's resources
/status/slaves/${key}/sysInfo
Number of cores in the slave machine
/status/slaves/${key}/sysInfo/cores
Available memory in the slave machine
/status/slaves/${key}/sysInfo/availableMemory
Free memory from available JVM memory
/status/slaves/${key}/sysInfo/freeMemory
The startup time of the slave
/status/slaves/${key}/sysInfo/startTime
The processes running on this slave
/status/slaves/${key}/processes
PID of the process. It's a string
/status/slaves/${key}/processes/${key}/pid
This denotes the type of process
/status/slaves/${key}/processes/${key}/type
This is the host on which process is running
/status/slaves/${key}/processes/${key}/host
Port on which this slave may be listening
/status/slaves/${key}/processes/${key}/port

Methods

Kills a process on this slave whose PID is sent
/exec/slaves/${key}/kill
